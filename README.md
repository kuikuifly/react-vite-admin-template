## 功能

```
- 登录 / 注销 / 注册 / 文章 / 用户

- 权限验证
  - 页面权限

- 全局功能
  - 动态侧边栏（支持多级路由嵌套）
  - 动态面包屑（支持自定义配置）

- 综合实例
  - 引导页
  - 组件示例
    - 编辑器
      - WangEditor
  - 页面示例
    - 文章管理
      - 文章列表
    - 角色管理
      - 用户列表
  - 权限验证
```

## 项目设置

### 一、Install dependencies,


```
$ pnpm install
```

 推荐使用 pnpm

### 二、Start the dev server,

```
$ pnpm run dev
```


### 三、Compiles and minifies for production

```
$ pnpm run build
```
